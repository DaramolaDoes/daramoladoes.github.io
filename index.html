Scientist Workflow

Reproducible User Path

  1. Select a governed dataset or upload a structured experimental file.
  2. Choose the validated method version and configure run options.
  3. Submit the workflow and capture a versioned execution record.
  4. Review outputs, quality-control flags, and explanation artifacts.
  5. Export results and attach run metadata to downstream analyses.
Agent Workflow

Autonomous System Consumption

  1. Discover the tool contract from a structured registry.
  2. Validate permission scope and allowed input types.
  3. Call the service using stable JSON inputs.
  4. Receive normalized outputs with provenance metadata.
  5. Trigger follow-on analysis only if guardrail checks pass.
